911 Emergency Call - Descriptive Analysis¶

Data Source: Kaggle.

Data Directory:

  • lat : Latitude
  • lng: Longitude
  • desc: Description of the Emergency Call
  • zip: Zipcode
  • title: Title
  • timeStamp: YYYY-MM-DD HH:MM:SS
  • twp: Township
  • addr: Address

Overview: A descriptive analysis on 911 Emergency Call Dataset.

Explore the Time Effect¶

In [12]:
# check the timeStamp data type
type(df['timeStamp'].iloc[0])
Out[12]:
str
In [13]:
# convert the string data to time data
df['timeStamp'] = pd.to_datetime(df['timeStamp'])
In [18]:
# Create 3 seperate column that capture the day of week, the month, and the exactly hour of time. 
df['Day of Week'] = df['timeStamp'].apply(lambda date:date.dayofweek)
df['Month'] = df['timeStamp'].apply(lambda date:date.month)
df['Hour'] = df['timeStamp'].apply(lambda date:date.hour)
dmap = {0:'Mon',1:'Tue',2:'Wed',3:'Thu',4:'Fri',5:'Sat',6:'Sun'}
df['Day of Week'] = df['Day of Week'].map(dmap)

See which day of week receive the call most frequently.
